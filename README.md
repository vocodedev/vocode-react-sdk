## Invalid Hook Call

You might have more than one copy of React in the same app when using React Hooks and NPM Link together.

Our solution
```
npm_link.sh
```

- [Invalid Hook Call Warning](https://legacy.reactjs.org/warnings/invalid-hook-call-warning.html)
